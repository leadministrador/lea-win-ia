# LEA WIN IA Integral
Incluye búsqueda por fecha, reuniones, carrera, participantes, datos del día, retiros,
campañas previas, videos opcionales, análisis 1.º–4.º, guardado e historial.

## Ejecutar
pip install -r requirements.txt
python app.py

## Videos dentro de la app
Agregar YOUTUBE_API_KEY en las variables de entorno.
Sin clave, abre una búsqueda pública de YouTube.

## Publicación
Compatible con Render, Railway y Docker.
